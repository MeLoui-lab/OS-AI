# OS-AI
Python project exploring local AI pipelines, voice processing and orchestration tools.


## Features
- Speech-to-text (Whisper)
- Text-to-speech (Qwen3-TTS, LuxTTS)
- Modular AI orchestration

## Structure
- main.py: main entry point
- dispatcher.py: orchestration logic
- Orchestrator/: core logic
- TTS/: text-to-speech modules
- Tools/: utilities
- Search/: search features with Ollama API
- Test/: experimental scripts

## Notes
Models, audio files and environment variables are not included in this repository.
